1. To what does a relative path refer?
Answer: A relative path shows the current directory of the file.

2. What does an absolute path start with your operating system?
Answer: An absolute path shows the root directory of the file.

3. What do the functions os.getcwd() and os.chdir() do?
Answer: os.getcwd() gives the path of the current working directory and os.chdir() changes the current working directory to the specified directory inside its argument.

4. What are the . and .. folders?
Answer: . folder is a relative directory while .. folder is a absolute directory.

5. In C:\bacon\eggs\spam.txt, which part is the dir name, and which part is the base name?
Answer: C:\bacon\eggs\ is the dir name and spam.txt is the base name.

6. What are the three “mode” arguments that can be passed to the open() function?
Answer: Three mode arguments that can be passed to the open() function is r for read, w for write and a for append.

7. What happens if an existing file is opened in write mode?
Answer: All its pre-existing data will be overwritten with the new inputs.

8. How do you tell the difference between read() and readlines()?
Answer: read() reads whole documents character by character and gives the total output if the reading size argument is blank.
readlines() reads whole documents line wise and returns a list of strings of each line along with the \n for encountering a newline.

9. What data structure does a shelf value resemble?
Answer: It resembles a dictionary value with keys() and values() method same as dictionary.
